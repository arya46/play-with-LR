Logistic Regression

Logistic Regression is one of the most common machine learning algorithms for classification. It a statistical model that uses a logistic function to model a binary dependent variable. In essence, it predicts the probability of an observation belonging to a certain class or label.

Assumptions
  • The labels are almost linearly separable.
  • The observations have to be independent of each other.
  • There is minimal or no multicollinearity among the independent variables.
  • The independent variables are linearly related to the log odds.

Advantages
  • Logistic Regression performs well when the dataset is linearly separable.
  • It does not require too many computational resources.
  • It is suitable for problems that have low latency constraints.

Disadvantages
  • Logistic Regression assumes that there is linearity between the dependent variable and the independent variables. In the real world, the data is rarely linearly separable
  • It is highly reliant on proper presentation of the data.
  • It is also not one of the most powerful algorithms out there. It can be easily outperformed by more complex ones.
